# Investa - Investment Preview UI Completion

## UI Only Implementation

Project: Investa

IMPORTANT:

Backend work is already completed.

Do NOT modify:

* Database
* Entities
* DTOs
* Controllers
* Services
* Business Logic
* APIs

This task is UI ONLY.

Use existing APIs and existing backend behavior.

---

# Context

The backend already supports:

* Learn More
* Investment Request workflow
* Approval workflow
* Rejection workflow
* Chat creation after approval
* Analytics tracking

However the Investment Preview page UX is incomplete.

The current screen does not clearly guide the investor through the decision-making process.

We need to improve the UI and user journey.

---

# Target Screens

Angular Client Portal

Primary Screen:

investment-preview.component

Only work on the UI layer.

---

# Business Goal

When an Investor opens:

Learn More

and reaches the Investment Preview page,

the page should clearly answer:

1. What is this opportunity?
2. Who is the founder?
3. Why should I care?
4. What can I do next?

The user should never feel lost.

---

# Page Structure

Reorganize the Investment Preview page into sections.

---

## Section 1 - Hero Area

Display:

* Cover Image
* Opportunity Name
* Investment Type
* Industry
* Founder Name
* Founder Reputation
* Status

Display key highlights.

Example:

Equity Investment

Technology

Founder Rating 4.8

Published

---

## Section 2 - Opportunity Summary

Display:

Short project overview.

Show:

* Problem
* Solution
* Market
* Key Value Proposition

Readable cards.

Not long paragraphs.

---

## Section 3 - Investment Information

Display according to investment type.

### Equity

Show:

* Share Price
* Total Shares
* Available Shares
* Minimum Participation

---

### Revenue Sharing

Show:

* Revenue Share %
* Target Amount

---

### Loan

Show:

* Interest Rate
* Duration
* Required Amount

---

### Founding Partner

Show:

* Required Skills
* Expectations

---

## Section 4 - Founder Information

Display:

* Founder Picture
* Founder Name
* Reputation
* Total Opportunities
* Active Opportunities

Add a button:

View Founder Profile

---

## Section 5 - Project Media

Display:

Photos

Videos

Documents

Media Count

Button:

View Project Media

Do NOT display huge galleries directly on the page.

Keep page clean.

---

Not at the very bottom.
